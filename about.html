I recently started working on bot detection at Datadome. Before joining Datadome as an R&D enginner, I was a PhD student at the University of Lille and Inria (France). My PhD focused on browser fingerprinting. I explored how it can be used for tracking, as well as to improve security on the web. In particular, a part of my work focused on using browser fingerprinting to detect bots and crawlers such as Chrome headless. Whenever possible, I try to open source the code I write on my Github profile.
