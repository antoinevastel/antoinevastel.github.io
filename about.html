I am currently a 3rd year PhD student at the University of Lille and INRIA (France). My research focuses on browser fingerprinting. I explore how it can be used for tracking, as well as to improve security on the web. In particular, my current work focuses on using browser fingerprinting to detect bots and crawlers such as Chrome headless. Whenever possible, I try to open source the code I write on my Github profile.
